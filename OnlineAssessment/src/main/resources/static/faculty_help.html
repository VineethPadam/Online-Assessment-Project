Faculty Help Guide

  • Upload the student details for your course using an .xlsx file.

    • The Excel file must be in the following format:
    student_roll_number student_name student_section student_year department student_email Password (Date of Birth)
    23691A05O2 Vineeth Padam D 3 CSE 23691A05O2@mits.ac.in 04-07-2005
    23691A05Q4 Yashwanth Reddy Kovvuru D 3 CSE 23691A05Q4@mits.ac.in 18-10-2004

  • Create new quizzes with questions and multiple options using the Add Quiz feature.

    1. Click on the Add Quiz button to start creating a new quiz.
    2. Enter the Quiz ID in the required format:

    Example Quiz IDs:

    • 2305-JAVA-01 → Same question paper for all sections in a department.
    • 2305A-FSDII-01 → Different question papers for different sections.

    Explanation of Quiz ID Format:

    • 23 – Joining year of the student batch.
    • 05 – Department code.
    • JAVA / FSDII – Subject name.
    • 01 – Assessment number (01 for Assessment-1, 02 for Assessment-2, etc.).
    • A – Section identifier (used only if different papers for each section).
    1. After entering the Quiz ID, click Submit.
    2. Choose the Excel file of the question paper to upload.
    3. The Excel must contain the following columns:
    Q_ID TEXT Option1 Option2 Option3 Option4 Correctoption
    2305A-JAVA-01-01 Which language is platform independent? Java C C++ Python option1
    2305A-JAVA-01-02 Select multiple OOP languages: Python Java C C++ option1,option2,option4

    Note:

    • For single correct answers, use format: option1.
    • For multiple correct answers, use format: option1,option3 (no spaces between commas).
    • Each question ID should follow the pattern 2305A-JAVA-01-01 (Quiz ID + Question No).

  • Activate the quiz for specific Department, Year, and Section so that only those students can attempt it.

    • Enter the Quiz ID you created earlier.
    • Select the corresponding Department, Year, and Section.
    • Click on the Activate button to enable the quiz for those students.

    Example:

    Quiz ID: 2305A-JAVA-01 → Activated for Department: CSE, Year: 3, Section: A

  • Faculty can view quiz performance and student scores after completion of the quiz.

    • Select the Quiz ID, Department, and Section.
    • Click View Results to display the marks of each student.
    • Click the Download button to export results in Excel format (.xlsx).

    Example Result Columns:

    • student_roll_number
    • student_name
    • Quiz Id
    • Score

  • Until results are published, students cannot view their marks or answer keys.

    • After verifying all submissions and marks, click Publish Result.
    • Once published, students can see their scores and correct answers.
    • Ensure the correct quiz is selected before publishing.
    We hope this guide helped you to efficiently upload student data, create and activate quizzes, and publish results using the MITS Assessment Portal.
